Pokedex App by Max IB

Uses PokeAPI to create an (infinte) scroll-style website that serves the
functionality of a basic pokedex. Link to PokeAPI:
https://pokeapi.co/

How to run:
Run "python manage.py runserver" in the root directory.

Interesting code:
views.py
homepage.html, homepage.css
pokemon_page.html, pokemon_page.css
